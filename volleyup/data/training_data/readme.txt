This folder contains a number of ball annotations for folders in the preceding directory

each entry contains data as follows

frame refers to the frame number in the respective image folder, i.e. 40 -> 40.jpg
x and y coords are coordinates for a circle with the ball 
rad is number of pixels to edge, center inclusive

bounding x and bounding y are coordinates for the top left point in the bounding rectangle around the ball
square length is the side length of the bounding rectangle
